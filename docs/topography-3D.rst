Topography
=================================

The following 3D topography functions are available:

.. list-table:: 
   :widths: 25 75
   :header-rows: 1
   
   * - Function
     - Description
   * - :doc:`ST_Drape`
     - Drape a geometry to a set of triangles
   * - :doc:`ST_TriangleAspect`
     - Return the aspect of a triangle
   * - :doc:`ST_TriangleContouring`
     - Split triangles into smaller triangles according to classes
   * - :doc:`ST_TriangleDirection`
     - Compute the direction of steepest ascent of a triangle
   * - :doc:`ST_TriangleSlope`
     - Compute the slope of a triangle as a percentage
   * - :doc:`ST_Voronoi`
     - Create a Vorono√Ø diagram
