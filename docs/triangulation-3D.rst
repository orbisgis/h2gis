Triangulation
=================================

The following triangulation functions are available:

.. list-table:: 
   :widths: 25 75
   :header-rows: 1
   
   * - Function
     - Description
   * - :doc:`ST_ConstrainedDelaunay`
     - Compute a constrained Delaunay triangulation based on a geometry
   * - :doc:`ST_Delaunay`
     - Compute a Delaunay triangulation based on points
   * - :doc:`ST_TESSELATE`
     - Tessellate a set of Polygon with adaptive triangles
