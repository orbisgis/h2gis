Measures
=================================

The following 3D measurements functions are available:

.. list-table:: 
   :widths: 25 75
   :header-rows: 1
   
   * - Function
     - Description
   * - :doc:`ST_3DArea`
     - Return a polygon's 3D area
   * - :doc:`ST_3DLength`
     - Return the 3D length of a Linestring
   * - :doc:`ST_3DPerimeter`
     - Return the 3D perimeter of a (multi)polygon
   * - :doc:`ST_SVF`
     - Compute the Sky View Factor
   * - :doc:`ST_SunPosition`
     - Compute the sun position
