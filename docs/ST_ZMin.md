# ST_ZMin

## Signature

```sql
DOUBLE ST_ZMin(GEOMETRY geom);
```

## Description

Returns the minimum z-value of `geom`.

## Example

```sql
SELECT ST_ZMin('LINESTRING(1 2 3, 4 5 6)');
-- Answer:    3.0
```
