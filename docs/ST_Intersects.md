# ST_Intersects

## Signature

```sql
BOOLEAN ST_Intersects(GEOMETRY geomA, GEOMETRY geomB);
```

## Description

Returns `TRUE` if `geomA` intersects `geomB`.

Intersects means that `geomA` and `geomB` have at least one point in common.
