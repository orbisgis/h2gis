# ST_ZMax

## Signature

```sql
DOUBLE ST_ZMax(GEOMETRY geom);
```

## Description

Returns the maximum z-value of `geom`.

## Example

```sql
SELECT ST_ZMax('LINESTRING(1 2 3, 4 5 6)');
-- Answer:    6.0
```
