# ST_XMax

## Signature

```sql
DOUBLE ST_XMax(GEOMETRY geom);
```

## Description

Returns the maximum x-value of `geom`.

## Example

```sql
SELECT ST_XMax('LINESTRING(1 2 3, 4 5 6)');
-- Answer:    4.0
```
